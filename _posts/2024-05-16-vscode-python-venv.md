# Steps

1. Ctrl + ` (Launch cmd)
2. ```python -m venv testenv```
3. Ctrl + Shift + p > Python: Select Interpreter > Choose a virtual environment
4. folder에 찾아가서 activate, deactivate script를 수행할 수도 있음
